This Website is developed using MERN Stack and Chakra UI for React.

Steps to run

Step1:
open cmd in frontend folder and type npm i
then 
open cmd in htvisionServer folder and type npm i

Step 2:
open cmd in htvisionServer folder and type nodemon server.js
then
open cmd in frontend folder and type npm start

Features:
1)Register User
2)Login User
3)Forget Password using mailtrap
4)Reset Password using mailtrap
5)Image upload using Cloudinary
6)Update profile
7)Change Password
8)Password Encryption in DB using Bcrypt Library
9)JWTokens
10)Change Profile Picture
11)Delete Profile
